# 《百論》卷一至卷十 · 中觀辯證全彙整

提婆菩薩造 · 婆藪開士釋 · 鳩摩羅什漢譯 · 中觀派 Madhyamaka

線上閱讀：<https://weiwen0925.github.io/bailun/>

## 內容

十品次第破除「我、一、異、情、塵、有果、無果、常、空」九種戲論執著：

| 卷 | 品 | 內容 |
|---|---|---|
| 一 | 捨罪福品 | 修行起點 · 止惡行善 · 最終捨福 |
| 二 | 破神品 | 破我執 · 解構神我（ātman） |
| 三 | 破一品 | 破「逼極成一」· 解構同一性 |
| 四 | 破異品 | 破「異 + 合」· 解構絕對差異性 |
| 五 | 破情品 | 破能取 · 解構認知主體（六根） |
| 六 | 破塵品 | 破所取 · 解構認知對象（六塵） |
| 七 | 破因中有果品 | 破數論派 satkāryavāda |
| 八 | 破因中無果品 | 破勝論派 asatkāryavāda |
| 九 | 破常品 | 破「無因常法」五座豐碑 |
| 十 | 破空品 | 破空執 · 空亦復空 |
